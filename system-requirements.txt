시스템 요구사항 (System Requirements)
작업시간 측정기 (Work Time Tracker) v1.2.0

─────────────────────────────────────────────────────────

[필수 요구사항]

운영체제:  Windows 10 64비트 이상 (Windows 10 / 11)
아키텍처:  x86-64 (64비트)
런타임:    Microsoft WebView2 Runtime
           - Windows 10 버전 1803 이상, Windows 11에는 기본 탑재
           - 미설치 시 설치 프로그램이 자동으로 다운로드 및 설치 진행

저장 공간:  설치 용량 약 10 MB
            데이터 파일 (time-tracker-data.json, time-tracker-history.json):
            수 KB 수준 (사용 기록에 따라 증가)

메모리:    최소 100 MB RAM (실사용 기준 50 MB 내외)


[선택 요구사항]

인터넷 연결:
  - 랭킹 기능 이용 시 필요 (Supabase 서버 통신)
  - 자동 업데이트 확인 시 필요 (GitHub Releases)
  - 핵심 작업시간 추적 기능은 오프라인에서도 정상 동작


[지원하지 않는 환경]

- Windows 7 / 8 / 8.1
- 32비트 운영체제
- macOS, Linux (Windows 전용 앱)


[참고]

- 시스템 트레이 아이콘 표시를 위해 작업 표시줄의 숨겨진 아이콘 영역이 활성화되어 있어야 합니다.
- Windows Defender 또는 일부 보안 소프트웨어가 첫 실행 시 경고를 표시할 수 있습니다.
  코드 서명 인증서가 적용되지 않은 프리웨어이므로 SmartScreen 경고가 나타날 수 있으며,
  "추가 정보 → 실행" 또는 "더 보기 → 어쨌든 실행"을 선택하면 정상 실행됩니다.
