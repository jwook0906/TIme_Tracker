# 설치 안내

## "Windows에서 PC를 보호했습니다" 경고가 뜨는 경우

설치 파일을 실행하면 Windows SmartScreen이 다음과 같은 경고를 표시할 수 있습니다.

> Windows에서 PC를 보호했습니다
> Microsoft Defender SmartScreen에서 인식할 수 없는 앱의 시작을 차단했습니다. 이 앱을 실행하면 PC가 위험할 수 있습니다.

**이 경고는 바이러스나 악성코드 때문이 아니라, 이 앱이 아직 상용 코드 서명 인증서로 서명되지 않은 소규모 개인 개발 프로젝트이기 때문에 나타납니다.** SmartScreen은 충분히 많은 사용자가 다운로드/실행한 적이 없는 프로그램에 대해 평판 정보가 없을 때 이런 경고를 표시합니다.

### 우회 방법

1. 경고 창에서 **"추가 정보"**(More info)를 클릭합니다.
2. 표시되는 앱 이름과 게시자를 확인한 뒤 **"실행"**(Run anyway) 버튼을 클릭합니다.

이 과정을 한 번만 거치면 정상적으로 설치 및 실행됩니다.

> 참고: 다운로드/실행 횟수가 누적되면 Microsoft의 SmartScreen 평판 시스템에 의해 경고 빈도가 자연스럽게 줄어듭니다. 추후 코드 서명 인증서를 적용하면 이 경고는 사라질 예정입니다.

## 설치 파일

GitHub Releases 페이지([releases](https://github.com/jwook0906/TIme_Tracker/releases))에서 최신 버전의 설치 파일(`.exe`, NSIS 인스톨러)을 내려받아 실행하세요. 앱은 실행 중 자동으로 새 버전을 확인하며, 새 버전이 있으면 알림을 통해 업데이트할 수 있습니다.
