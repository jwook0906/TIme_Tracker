개인정보처리방침 (Privacy Policy)
작업시간 측정기 (Work Time Tracker)
최종 수정일: 2026-06-09

─────────────────────────────────────────────────────────

1. 개요

작업시간 측정기(이하 "앱")는 사용자의 업무 시간을 측정하고 관리하기 위한 데스크탑 애플리케이션입니다.
본 방침은 앱이 수집·사용·보관하는 정보에 대해 설명합니다.


2. 수집하는 정보

[로컬에서만 수집되는 정보]
- 사용자가 추적 대상으로 등록한 프로그램의 이름 및 사용 시간
- 현재 활성화된 프로세스 이름 (추적 여부 판단 목적으로만 사용, 외부 전송 없음)
- 유휴(idle) 시간
- 날짜별 작업 기록

위 정보는 사용자의 기기 내 파일(time-tracker-data.json, time-tracker-history.json)에만 저장되며,
개발자 또는 제3자에게 전송되지 않습니다.

[랭킹 기능 이용 시 수집되는 정보]
랭킹 기능은 선택 사항이며, 참여를 선택한 경우에 한해 아래 정보가 수집됩니다.

- 익명 계정 식별자 (Supabase 익명 인증 발급, 이메일·비밀번호 불필요)
- 사용자가 직접 입력한 닉네임
- 날짜별 총 작업시간 합계 (초 단위, 프로그램 세부 내역은 전송되지 않음)
- 연속 작업일수 (streak)

위 정보는 Supabase(supabase.com) 서버에 저장됩니다.
닉네임과 작업시간 합계는 랭킹 화면에 다른 참여자에게 공개됩니다.


3. 정보 사용 목적

- 로컬 수집 정보: 사용자 본인의 작업시간 통계 표시 전용
- 랭킹 정보: 전체 참여자 순위 표시 전용


4. 정보 보관 및 삭제

[로컬 데이터]
- 기기 내 파일에 보관되며 사용자가 직접 파일을 삭제하여 제거할 수 있습니다.
- 앱 내 개발자 탭(F11+F12)의 "기록 초기화" 기능으로 작업 기록을 삭제할 수 있습니다.

[랭킹 데이터]
- 앱 내 랭킹 탭의 "개발자 모드 → 내 랭킹 기록 초기화" 기능으로 본인 데이터를 0으로 초기화할 수 있습니다.
- 익명 계정은 앱을 재설치하거나 세션을 삭제하면 더 이상 접근할 수 없게 됩니다.


5. 제3자 제공

수집된 정보는 다음의 경우를 제외하고 제3자에게 제공되지 않습니다.

- Supabase: 랭킹 기능 운영을 위한 데이터 저장 (위탁)
  Supabase 개인정보처리방침: https://supabase.com/privacy

그 외 광고, 마케팅, 분석 목적의 제3자 제공은 없습니다.


6. 아동 개인정보

본 앱은 만 14세 미만 아동을 대상으로 하지 않으며, 의도적으로 아동의 개인정보를 수집하지 않습니다.


7. 방침 변경

본 개인정보처리방침이 변경될 경우 앱 업데이트 릴리즈 노트에 변경 사항을 공지합니다.
